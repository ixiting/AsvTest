AsvTest — лёгкое консольное приложение, предназначенное для демонстрации работы с библиотекой Asv.Mavlink.
Приложение обеспечивает отображение телеметрии летательного аппарата (широта, долгота, абсолютная и относительная высота, вертикальная скорость) в реальном времени,
а также отправку базовых управляющих команд (взлёт, посадка, переход в указанную точку координат) на аппарат (SITL).

Основные возможности
- Отображение телеметрии в реальном времени в компактном формате консольной таблицы (на основе Spectre.Console).
- Реактивная обработка телеметрической информации (R3/System.Reactive).
- Реализация команд взлёта и посадки, и движение к точке посредством микросервиса Asv.Mavlink.

Предварительные условия для запуска (Windows)
- Установленный .NET 8 SDK.
- Запущенный экземпляр SITL, предоставляющий подключение MAVLink по TCP на адрес 127.0.0.1:5760 (используемое проектом значение по умолчанию).

Запуск SITL (пример)
- Вы можете использовать любой доступный SITL. Критически важно, чтобы автопилот был доступен по TCP-порту 5760.

Сборка и выполнение
```powershell
dotnet build
dotnet run --project AsvTest/AsvTest.csproj
```
Интерактивное управление
Controls (interactive)
- t — выполнить взлёт (высота по умолчанию: 10 м).
- l — выполнить посадку.
- g — переход в указанную точку (программа запросит широту, долготу и опционально высоту).
- q — завершение работы приложения.

Диагностика и устранение неполадок
- В случае превышения времени ожидания подключения убедитесь, что SITL запущен, доступен по адресу 127.0.0.1:5760, и что сетевые правила не блокируют соединение.
- Файл commands.log содержит сведения об ответах на команды (ACK) и стеки исключений, что позволяет проводить анализ отказов управления.
